Welcome to the Abyssal Index

The front door to the Abyssal Index — a structured knowledge system for raw thoughts, theories, and projects.

wiki index welcome

The Abyssal Index is a personal knowledge system built on a simple premise: every thought deserves a place, and no thought should be lost.

What This Is

A structured vault of ideas, theories, research, and projects — captured raw, refined over time, and connected by the patterns that emerge between them. Not a blog. Not a wiki in the traditional sense. A living index of one mind's working memory, made visible.

The Floor System

Content moves through five floors as it matures:

  • B2 — The Void. Where every thought enters. No friction, no judgment. Capture first, organize later.
  • B1 — The Workshop. Ideas getting shaped. Tagged, categorized, starting to connect.
  • G — Ground Floor. Structured and useful. Ready for reference.
  • 1 — The Library. Refined knowledge. Well-connected, well-written.
  • 2 — The Gallery. Published. What you're reading now.

Domains

The index organizes content into seven domains:

  • Theories — philosophical frameworks, mental models, speculative ideas
  • Research — investigations, literature notes, technical deep-dives
  • Projects — active builds, architecture decisions, progress logs
  • Systems — tools, workflows, infrastructure, process design
  • Creative — fiction, worldbuilding, game design, art direction
  • Personal — reflections, journal-style entries, life observations
  • Reference — how-to guides, cheat sheets, canonical docs (like this page)

How It Works

The Abyssal Index runs on a Rust core library with SQLite full-text search. Content lives as plain Markdown files with YAML frontmatter — no vendor lock-in, no proprietary format. AI assists with triage (auto-categorization, tagging, summarization) but never overwrites human decisions.

Desktop access is through Brain Dump PC. Mobile capture through Brain Dump for Android. Claude reads the index via MCP. Everything connects, nothing requires the other to function.

Built From the Deep End

This system exists because existing tools either demanded too much structure upfront (killing capture momentum) or too little structure ever (creating searchable chaos). The Abyssal Index is the middle path: zero friction on entry, progressive refinement by design.

Every page you see here earned its way to the Gallery.
